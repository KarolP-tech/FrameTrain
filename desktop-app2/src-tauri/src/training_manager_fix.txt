In der Funktion create_new_model_version müssen folgende 3 Stellen geändert werden:

Zeile ca. 900:
VORHER: [&resolved_model_id],
NACHHER: [model_id],

Zeile ca. 906:
VORHER: let version_path = models_dir.join(&resolved_model_id).join("versions").join(&version_id);
NACHHER: let version_path = models_dir.join(model_id).join("versions").join(&version_id);

Zeile ca. 927:
VORHER: &resolved_model_id,  // Use resolved_model_id instead of model_id
NACHHER: model_id,

Das Problem: `resolved_model_id` ist eine lokale Variable in der `start_training` Funktion, 
aber sie existiert nicht in der `create_new_model_version` Funktion. 
Die Funktion erhält bereits den korrekten `model_id` als Parameter.
